# Моделирование процесса заказа в интернет-магазине

## Описание проекта
Проект выполнен в рамках подготовки к стажировке системного аналитика. Смоделирован полный жизненный цикл заказа — от добавления товара в корзину до доставки клиенту.

## Артефакты
- [BPMN-диаграмма](BPMN-диаграмма.png) — основной процесс с ветвлениями (наличие товара, повреждение на складе)
- [UML Use Case Diagram](UML-диаграмма.png) — роли участников: клиент, менеджер склада, курьер, платежный шлюз
- [Текстовые сценарии](scenarios.docx) — основной и 2 альтернативных сценария (товара нет, отмена заказа)

## Используемые нотации
- BPMN 2.0
- UML (Use Case)

## Контекст
Студент 2-го курса Бауманского университета (ИУ-5). Проект показывает понимание бизнес-процессов и навыки моделирования.
